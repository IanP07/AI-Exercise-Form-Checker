## How to run our app:

cd to the react project directory (/ai-form-checker)

Then run ```npm install```, followed by:

```npm install @mediapipe/pose @mediapipe/drawing_utils```

And finally, run:

```npm run dev```

This should display two links. The first will work on your local device, the second will work on any other device, like a mobile phone
